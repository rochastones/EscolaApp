# EscolaApp
O EscolaApp é um aplicativo móvel desenvolvido em Kotlin com Jetpack Compose para gerenciamento de alunos, disciplinas, notas e professores. 
O projeto foi idealizado inicialmente com o objetivo de aprimorar o conhecimento em aplicações Android, colocando em prática os recursos e conceitos da linguagem Kotlin para a criação de soluções digitais baseadas em boas práticas de programação e regras de negócio, capazes de atender a demandas reais da sociedade.
1. Arquitetura:
•	Arquitetura Limpa (Clean Architecture) com separação de camadas
•	Padrão MVVM (Model-View-ViewModel)
•	Múltiplas fontes de dados (Local, API REST, Nuvem)
•	Troca dinâmica entre fontes de dados
•	Segurança com autenticação e JWT
•	Experiência do usuário profissional

2. Conceito do APP
2.1 Funcionalidade Principal
O app permite o gerenciamento completo de alunos, com operações de:
•	Inserir novos alunos
•	Listar alunos cadastrados
•	Editar informações dos alunos
•	Excluir alunos

3.Troca Dinâmica de Fonte
O usuário pode alternar entre as 3 fontes de dados em tempo real através da tela de configurações:
Configurações → Selecionar Fonte de Dados
Local (Room)
API REST (PHP/MySQL)
Firebase Firestore

4. Aplicações do Projeto
4.1 Utilização Acadêmica
O app pode ser utilizado para:
•	Gerenciar alunos em escolas/cursos
•	Controlar notas e aprovações
•	Manter histórico de alunos

5. Utilização Didática
O código serve como referência para:
•	Clean Architecture em Android
•	MVVM com Compose
•	Múltiplas fontes de dados
•	Injeção de dependência
•	Segurança em apps
